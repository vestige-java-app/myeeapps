This 2 EAR applications can be installed on a TomEE running on Vestige running on a JVM 8

You can verify that WAR are correctly handle by Tomcat :
- <http://localhost:8080/mywar/hello>

And then that EAR is working with TomEE :
- <http://localhost:8080/war1/hello>
- <http://localhost:8080/war2/hello>
